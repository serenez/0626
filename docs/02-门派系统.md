# 02 · 门派系统

> 六大门派为本作自定义设定；部分世界观与师门任务参考 `design-source/XK策划案.xlsx`（《侠客MUD》老案）。
>
> **本文为骨架**。门派世界观、弟子阶位、师门任务框架已落定；具体**武学数值、技能树、师门任务数值奖励**属于后续《武学系统》《任务系统》，本文只列接口与待补清单。

## 1. 概述

门派是玩家在游戏中加入的核心组织，是探索、成长和挑战的主线载体。

- **首发开放 6 个初始门派**：少林寺、太玄教、青岚剑宗、万花谷、唐门、刀宗。
- **加入限制**：少林寺**只招男性**，其余门派不限性别。
- **赛季制关联**：暂空，后续设计。

### 1.1 门派职能配置

六大门派覆盖坦克 / 输出 / 治疗 / 增益四类职能，保证组队（≤3 人小队）可凑出标准阵容。

| 门派 | 武器 | 核心职能 | 一句话定位 |
|------|------|----------|-----------|
| 少林寺 | 拳 / 棍 | 🛡 坦克（肉） | 禅武合一，高生存抗压前排 |
| 太玄教 | 剑 | ✨ 增益（buff） | 道门符箓，增益己方 / 削弱敌方 |
| 青岚剑宗 | 剑 | ⚔ 输出（DPS） | 快剑高爆发，纯粹剑道输出 |
| 万花谷 | 鞭 | ❤ 治疗（奶） | 医毒花艺，疗伤续航后排 |
| 唐门 | 暗器 | ☠ 输出（毒/DPS） | 暗器加毒，持续 + 爆发 |
| 刀宗 | 刀 | 💥 输出（爆发DPS） | 高暴击、刀意叠加越战越强 |

> 门派基础一览见 [`data/门派一览.csv`](../data/门派一览.csv)。

## 2. 通用机制（师门系统）

所有门派共享同一套拜师、晋阶、判门、学艺规则，差异只在门派任务内容与武学。

### 2.1 弟子阶位

| 阶位 | 师门任务上限/日 | 任务侧重 |
|------|:---------------:|----------|
| 入门弟子 | 20 个/日 | 送货、打杂 |
| 一阶弟子 | 10 个/日 | 采买材料、PVE 野怪 |
| 二阶弟子 | 5 个/日 | 少量 PVP 任务 |
| 三阶弟子 | 2 个/日 | 以 PVP 任务为主 |

> 晋阶顺序固定：入门 → 一阶 → 二阶 → 三阶；满足条件（门派好感度、贡献度）即可晋升。

### 2.2 拜师

- 现有 6 个门派，满足条件即可拜师，拜师顺序非固定。
- 入门即增加对应门派**好感度 +100**。
- 性别限制：少林只招男性。

### 2.3 判门

| 方式 | 触发 | 后果 |
|------|------|------|
| 主动判门 | 玩家向当前师父提出请求，需满足一定条件 | 脱离门派 |
| 自动判门 | 与所在门派好感度降为负数 | 自动判门，且 48 小时内有几率遭门派暗杀（清理门户） |

### 2.4 学习武学

#### 武学品阶

门派武学按强度分四阶，品阶决定**解锁方式**与是否需要秘籍。

| 品阶 | 定位 | 解锁方式 | 是否需秘籍 |
|------|------|----------|:----------:|
| 基础武学 | 入门基本功 | 潜能直接学 | 否 |
| 进阶武学 | 成长期主力武学 | 潜能直接学 | 否 |
| 上乘武学 | 高级武学 | 贡献购秘籍解锁上限 + 潜能学 | ✅ |
| 镇派武学 | 门派最强武学 | 贡献购秘籍解锁上限 + 潜能学（要求更高） | ✅ |

> 一般**上乘武学起**才需要秘籍解锁等级上限；基础 / 进阶武学花潜能即可直接学习。

#### 两种学习方式

- **方式一 · 潜能直学**（基础 / 进阶武学）
  花费**潜能**向当前师父学习对应武学，无需秘籍。

- **方式二 · 秘籍解锁 + 潜能学**（上乘 / 镇派武学）
  1. 先花费**门派贡献**购买该武学的**秘籍 / 书籍**（一般分**上、中、下三册**）；
  2. 每册解锁对应武学的**一段等级上限**（上册 → 中册 → 下册，逐段提升）；
  3. 再花费**潜能**向当前师父学习，最高只能学到已解锁的等级上限。

#### 贡献来源

门派贡献通过**门派每日任务**及其他渠道（势力、活动等）获取，用于购买武学秘籍及门派内消费。

#### 武学等级上限来源

一门武学（尤其上乘 / 镇派）的可学等级同时受多个上限约束，**实际可学到的等级取其中最低值**：

| 上限 | 来源 | 作用 |
|------|------|------|
| 秘籍解锁上限 | 花门派贡献购上 / 中 / 下三册逐段解锁 | 该武学的**理论天花板**，没买到就练不上去 |
| 师父可教上限 | 师父的「可教等级」属性 | 师父能带你到的上限；**通常低于师父自身等级**（师父留一手） |
| 自修可达上限 | 读书、实战顿悟等自修途径 | 脱离师父后继续突破的途径 |

> **当前可学等级 = min( 秘籍解锁上限 , max( 师父可教上限 , 自修可达上限 ) )**
>
> 说人话：**秘籍是天花板**；在天花板之下，**师父教**和**自己修**是两条并行的抬高途径，取较高的那条。

#### 师父可教上限与自修

- 师父有两个独立属性：**自身等级上限** 与 **可教等级上限**；后者一般 **低于** 前者——越高品阶的武学，师父越「教不全」，差值越大。
- 当玩家对某武学的可学需求**超过师父的可教上限**后，**无法再通过该师父继续学习此武学**。
- 此后改走**自修途径**继续突破，直至顶到秘籍解锁上限：
  - **读书**：研读对应武学书籍 / 心得（消耗道具）。
  - **实战顿悟**：在实战中按一定条件 / 概率领悟提升。
- 自修通道**仅在师父可教上限封顶后开放**，定位为后期长线成长，**不替代**师父的前中期教学。

#### 通用约束

- 学习前需满足门派**好感度 + 贡献度**要求。
- 三道门槛缺一不可：**秘籍解锁上限**（天花板）、**师父可教 / 自修**（抬高途径）、**潜能**（填充等级）。
- 玩家应能清晰看到当前卡在哪个上限（缺秘籍 / 师父已封顶待自修 / 缺潜能），避免「学不了却不知为何」的体验。

### 2.5 师门任务设计原则

- 全手动完成（非挂机）。
- 任务种类多样，各门派内容不同，不同阶位获取的任务不同。
- 奖励定位：中量潜能、中量经验、材料 / 银两来源、少量黄金（具体数值见后续《任务系统》）。

## 3. 初始门派

> 每个门派结构统一：定位 → 武器 → 武学体系 → 特色 → 师门任务。武学的**具体招式数值与技能树**留待《武学系统》补全，此处只列名称与定位。

### 3.1 少林寺 🛡 坦克

- **定位**：禅武合一，内外兼修、刚柔相济；武林泰山北斗。前排肉盾，高生存抗压。**只招男性。**
- **主要武器**：拳套、棍。
- **武学体系**：少林长拳（基础刚猛拳法）、少林棍法（招牌）、达摩易筋经（内功巅峰，气血 / 抗性核心）、金钟罩 / 铁布衫（减伤护体）、少林七十二绝技（绝技总称）。
- **特色**：禅武合一，修武同时研读佛经；以厚血、减伤、嘲讽抗压为战斗核心。

| 师门任务 | 内容 |
|----------|------|
| 宣扬佛法 | 去指定寺庙宣扬佛法，有几率触发战斗，胜利或宣扬完毕后回禀师父 |
| 挑水 | 领水桶到水井打两桶水（一次一桶）交给烧火僧 |
| 敲钟 | 钟楼敲钟三次，全少林可闻；为听到钟声的少林玩家加 30 分钟防御 +2% buff |
| 砍柴 | 松树林砍木 → 木墩劈柴 → 交烧火僧 |
| 炼药 | 药王殿找明斛领材料，丹炉炼活血散后交付 |
| 抓捕叛僧 | 指定地图击败叛僧交戒律院；有几率出现放人选项（得银两+神秘道具，但任务失败） |
| 门派挑战 | 出门击败挑衅者 |

### 3.2 太玄教 ✨ 增益

- **定位**：道门剑修，阴阳五行、符箓御剑；以**增益己方、削弱敌方**为核心的辅助门派。
- **主要武器**：剑（太玄剑 / 太极双剑）。
- **武学体系**：玄门剑法（基础剑法）、五行符箓（群体增益 / 减益 buff）、太极护体（己方减伤）、乾坤御剑术（御剑远攻）、道心通明（修行境界，强化增益效果）。
- **特色**：内外兼修、身心合一；战斗中靠 buff / debuff 调控战局，自身有一定输出与自保。

| 师门任务 | 内容 |
|----------|------|
| 炼药 | 炼丹房领材料 → 炼丹炉炼丹 → 交付 |
| 送药 | 领药包，限时送达后返回提交 |
| 救援 | 寻找遇困同门，战斗胜利后回师门交差 |
| 巡检 | 指定范围走动，随机触发与窃贼战斗，胜利交差 |
| 杂事 | 打扫演武坪 / 擦拭三清殿祖师像 / 整理藏经阁 / 给药草园施肥 |
| 挑战 | 解剑岩教训闹事者，限时十分钟，打赢交差 |
| 修缮 | 库房领材料，到指定地点（廊桥 / 闻道亭 / 五行宫）修整 |
| 设坛祈福 | 在三清殿设坛，为同门施放限时增益 buff |

### 3.3 青岚剑宗 ⚔ 输出

- **定位**：以剑为主、快剑见长的正道剑派；纯粹的**高爆发剑道输出**。
- **主要武器**：剑。
- **武学体系**：青岚快剑（以快制敌，连击）、追风十三剑（爆发连招）、剑意合一（叠加剑意提升伤害与会心）、御风身法（高机动近身）。
- **特色**：身法灵动、剑法连贯，以会心 / 连击堆叠爆发；护甲薄，重走位与切入时机。

| 师门任务 | 内容 |
|----------|------|
| 酿酒原料 | 鸣弦泉打一桶水酿酒（骑马会洒水） |
| 清理叛徒 | 按指示地点击杀剑宗叛徒 |
| 维持安宁 | 山脚松林解决伤人猛虎 |
| 论剑 | 演武场与同门切磋，胜场达标后交差 |
| 巡山 | 巡查青岚山数处哨点，随机遭遇山匪，击退后回报 |

### 3.4 万花谷 ❤ 治疗

- **定位**：隐于花海的医毒花艺一脉，精研医术、药理、音律、花卉；以**鞭为兵、以医为本**的后排治疗（奶）门派。
- **主要武器**：软鞭 / 花鞭。
- **武学体系**：回春疗伤诀（单体 / 群体治疗核心）、百花缭乱鞭法（鞭法输出 + 控制）、解毒清心（驱散异常状态）、护花铃（团队减伤光环）、医毒同源（治疗兼带毒系辅助输出）。
- **特色**：治疗续航 + 状态驱散为核心，鞭法提供远距离骚扰与控制；自身脆弱，依赖走位与队友保护。

| 师门任务 | 内容 |
|----------|------|
| 采药 | 到药谷采集指定草药，限定数量后交付药师 |
| 制药 | 药庐领材料，按配方炼制疗伤丹 / 解毒散后交付 |
| 救治伤员 | 前往指定地点为受伤村民 / 同门施救（治疗读条），完成后回报 |
| 打理花圃 | 浇灌、除虫、修剪谷中花圃（随机地点） |
| 抚琴静心 | 在听雨阁抚琴一曲，为在场同门施放限时回血 buff |
| 驱赶采花贼 | 谷中混入采花 / 盗药之徒，找出并击退 |
| 护送药商 | 护送采买药材的商队往返，途中可能遭劫 |

### 3.5 唐门 ☠ 输出（毒）

- **定位**：巴蜀暗器世家，机关 + 毒术，以最小之力制敌于无形；**持续毒伤 + 暗器爆发**双形态输出。**世袭传承，要职唐姓。**
- **主要武器**：飞镖 / 暗器。
- **武学体系**：爆裂镖（击中爆炸，爆发）、毒镖（麻痹 / 剧毒，持续伤害）、漫天花雨（群体暗器）、轻功身法（拉开距离风筝）、毒术研究（毒经阁，强化中毒效果）。
- **特色**：远程输出，毒系 DoT 叠加 + 暗器爆发；近身能力弱，靠走位与控制保持距离。

| 师门任务 | 内容 |
|----------|------|
| 终止谣言 | 唐家集击杀散布谣言者，回禀师父 |
| 排除危险 | 击杀潜入绝情竹海的窥探者并获取目的 |
| 寻回物品 | 找回被盗的绝情散炼制图谱 |
| 炼制毒药 | 药田采腐心草 + 唐景仁处领寒蜈蜕 → 炼断肠散交师父 |
| 打理药田 | 唐云逸处接任务，按要求浇灌药丛 |

### 3.6 刀宗 💥 输出（爆发）

- **定位**：红枫镇刀法一脉，性格豪迈、快意恩仇；**高暴击、命中偏低**的爆发型输出。
- **主要武器**：刀。
- **武学体系**：狂风快刀（速度连斩）、霸王斩（力沉高爆发）、刀宗三式（**累积刀意**，越战越强分层 buff）、影踪刀法（潜行切入）。
- **特色**：以刀意叠加换取持续走高的爆发，越战越强；命中偏低需以暴击与刀意补偿，重连续作战。

| 师门任务 | 内容 |
|----------|------|
| 维持安宁 | 藏锋集劝说 / 教训闹事小混混 |
| 清理叛徒 | 按指示地点击杀刀宗叛徒 |
| 运送铁胚 | 铁火堂刀破穹领 8 个玄铁铁胚送各地打铁铺 |
| 救援同门 | 解救被劫持的运矿弟子，回师门报告 |
| 查明情况 | 红枫谷探查可疑江湖客，击败取情报拼凑真相 |
| 帮助总管 | 大摆筵席数酒 / 闻香院接樱花酿酒（读条到"马上落地"时伸手） |
| 读书答题 | 凌霄阁读书 → 回总管处答题 |
| 同门过招 | 与同门切磋（核心弟子需与凌霄阁顶层两人切磋） |
| 击败挑战者 | 门外击败其他门派挑战者 |
| 门派巡逻 | 巡逻十个地点找小偷，概率遇到打一架 |
